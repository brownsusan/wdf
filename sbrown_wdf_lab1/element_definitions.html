h1
h1 is used for headers with the most importance. ex: <h1>Sample Header</h1>
h2
h2 is used for headers with secondary importance. ex: <h2>Sample Header</h2>
h3
h3 is used for headers with less importance than the second header. ex: <h3>Sample Header</h3>
p
p is used to start a paragraph. ex: <p>This is a sample paragraph.</p>
ul
ul is used to start an unordered list with bullet points. ex: <ul><li>Unordered List item 1</li></ul>
ol
ol is used to start an ordered list with numbered items. ex: <ol><li>Ordered List item 1</li></ol>
li
li is used to display a single list item. ex: <li>List item 1</li>
