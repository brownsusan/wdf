b & i

With the b element keywords, product names, and speakers can be emphasized. Using the i element, technical terms, phrases from another language, or thoughts can be represented in an alternate voice.

Code

In order to represent computer code the code element is used.

del & ins

When revising documents, removed content can be accurately represented with the del element. When adding content in to the document the ins element is used.

dfn

To represent a definition, the dfn element is used.

Figure & Figcaption

hr

this paragpah is being broken up by the hr element.

The hr element is used to represent a thematic break, or the end of one topic and the start of another.

kbd

kbd represent user input. It is usually used for keyboard commands such as pressing Command+Shift+a to select all.

mark

The mark element is used to highlight text.

meter

The meter element is used to depict something that is represented on a scale. For example, if a student scores 80 out of 100 on a test, their score would be 80 out of 100

pre

The pre element represents blocks of pre-formatted content. Ex: 

    
		        test         these
		words       display     pre-formatting.
		        word.   word
		word.              word.

samp

samp represents sample output from a program such as an error message. Ex: Error: This is an error.

small

The small element is used for side comments such as copyright information and disclaimers. It is rendered in smaller font-size by default.

sup

The sup element is used to apply superscript (superscript) to text.

sub

Just as sup applies superscript, sub applies subscript (subscript)

time

The time element represents either time or a precise date. If a person posted an article at February, 19th 2:15 p.m then they would use the time tag.

var

The var element represents variables in a paragraph. If there are n pipes leading to the ice cream factory then I expect at least n flavors of ice cream to be available for purchase!
